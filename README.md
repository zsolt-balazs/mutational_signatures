# Mutational_signatures
Lab practice on somatic mutations and mutational signatures in cancer

The respository contains a Jupyter notebook that can for example be run by using Google Colab. The underlying data are found in the `data` folder.

The data were generated by randomly sampling 100 patients from each cancer type in the TCGA dataset. The data for today's class was generated using the R package sigminer and the class was inspired by the tutorial: https://shixiangwang.github.io/sigminer-book/basic-workflow.html

The code was generated using Gemini 3.
